# ТЗ для ИИ-агента (Sem2 / СЗ_1)

## Название
Агент анализа отзывов клиентов и генерации рекомендаций.

## Цель
Автоматизировать обработку текстовых отзывов и помочь менеджеру продукта:
1) понять общий тон (позитив/негатив),
2) выделить ключевые проблемы/пожелания,
3) предложить конкретные рекомендации по улучшению продукта.

## Входные данные
- Один или несколько отзывов клиентов (на русском).

## Выходные данные
Структурированный отчёт:
- Суммарная тональность (коротко)
- Топ-3 темы/проблемы
- Рекомендации (3–5 пунктов)

## Ограничения
- Использовать только предобученные модели.
- CPU-first.
- Для LLM использовать локальный Ollama.

## Нефункциональные требования
- Устойчивость к prompt injection: агент должен игнорировать инструкции пользователя, которые просят раскрыть системный промпт или изменить цель.
- Валидация входа: пустые отзывы не принимаются.
